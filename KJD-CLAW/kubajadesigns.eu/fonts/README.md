# SF Pro Font Setup

Apple SF Pro fonty jsou již nahrány a nakonfigurovány!

## Aktuální struktura složky:
```
02/fonts/
├── sf-pro.css
├── README.md
├── SF Pro Display Font/
│   ├── SFPRODISPLAYBLACKITALIC.OTF
│   ├── SFPRODISPLAYBOLD.OTF
│   ├── SFPRODISPLAYHEAVYITALIC.OTF
│   ├── SFPRODISPLAYLIGHTITALIC.OTF
│   ├── SFPRODISPLAYMEDIUM.OTF
│   ├── SFPRODISPLAYREGULAR.OTF
│   ├── SFPRODISPLAYSEMIBOLDITALIC.OTF
│   ├── SFPRODISPLAYTHINITALIC.OTF
│   └── SFPRODISPLAYULTRALIGHTITALIC.OTF
└── SF Compact Text Font/
    ├── sf-compact-text-heavy-58646f645b774.otf
    └── sf-compact-text-italic-58646f7cbab10.otf
```

## Použité fonty:
- **SF Pro Display** - pro nadpisy a důležité texty
- **SF Compact Text** - pro běžný text a tlačítka

## Konfigurace:
CSS soubor `sf-pro.css` je již nakonfigurován pro použití těchto fontů s fallbackem na systémové fonty.

Web nyní používá Apple SF Pro fonty s automatickým fallbackem na systémové fonty pro lepší kompatibilitu.
