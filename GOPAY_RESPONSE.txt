Vážený pane Sternade,

děkuji za upozornění na chybějící ověření stavu platby.

**Problém byl identifikován a opraven:**

1. **Chyba:** Notification URL `gopay_notify.php` neexistovala - soubor byl nyní vytvořen
2. **Oprava:** Implementoval jsem správný proces podle dokumentace https://doc.gopay.cz/#odeslani-notifikace
3. **Ověření:** Po přijetí HTTP notifikace nyní VŽDY provádíme dotaz na stav platby přes REST API metodu `GET /payments/payment/{id}`

**Implementace (`gopay_notify.php`):**
- Příjem HTTP notifikace (GET/POST)
- **Okamžitý dotaz na stav platby** přes GoPay REST API
- Zpracování stavu (PAID, CANCELED, TIMEOUTED, REFUNDED)
- Vrácení HTTP 200 OK

**Provedl jsem novou testovací platbu:**
- Platba bude provedena do několika minut
- Notification URL: `https://kubajadesigns.eu/gopay_notify.php`

Prosím o potvrzení, že notifikace jsou nyní správně zpracovány.
